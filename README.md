# 说明文档

## codecli: 帮助你快速搭建和开发前端项目的 CLI

> 想不起来其他名字，以这个命名吧~

如何安装？

```
npm install codecli -g

```

### 目前支持 Vue，后期会新增 React，Angular 相关模版，尽情期待~

<hr>

如何使用？

1，简单易上手，创建项目命令

```
codecli create your_project_name
```

2,自动拉取项目模板、安装项目依赖、打开浏览器 http://localhost:8080/、自动启动项目
